# 探索

对于冒险者的成功而言，没有多少规则比与视觉、照明以及如何破坏物体相关的规则更为重要。以下是每个规则的详细说明。

## 视觉与光照

:::origin[[英文原文](https://aonprd.com/Rules.aspx?ID=50)]
:::

矮人和半兽人拥有黑暗视觉，但第2章中介绍的其他种族需要光源来看到周围的环境。见表7-10了解光源照亮的半径及其持续时间。表中的“延伸区域”一栏表示在光照半径之外的区域，光照等级提高一级（例如从黑暗变为昏暗）。

| 物品 | 照明半径 | 延伸区域 | 持续时间 |
| :---- | :----: | :----: | :----: |
| 蜡烛 | n/a<sup>1</sup> | 5尺 | 1小时 |
| 永燃火把 | 20尺 | 40尺 | 永久 |
| 普通提灯 | 15尺 | 30尺 | 6小时/品脱 |

| 日光杖 | 30尺 | 60尺 | 6小时 |

| 法术 | 照明半径 | 延伸区域 | 持续时间 |
| :---- | :----: | :----: | :----: |

在明亮光照区域内，所有角色都能清晰地看到周围环境。某些生物，如有强光敏感或强光目盲的生物，在明亮光照区域内会受到惩罚。在明亮光照区域内，生物无法使用隐匿技能，除非它是隐形的或有掩蔽。明亮光照区域包括在阳光直射的室外或处于昼明术范围内的区域。

普通光照的功能与明亮光照相同，但有强光敏感和强光目盲的角色不会受到惩罚。普通光照区域包括白天在森林树冠下的区域、在火把20尺范围内的区域，以及光亮术范围内的区域。

在昏暗光照区域内，角色可以勉强看清楚一些东西。生物在此区域内对没有黑暗视觉或在黑暗中视物能力的生物具有隐蔽（战斗中有20%失手率）。在昏暗光照区域内的生物可以进行隐匿检定来隐藏自己。昏暗光照区域包括夜晚有月亮的户外区域、明亮的星光下，以及在火把20到40尺之间的区域。

在黑暗区域内，没有黑暗视觉的生物实际上相当于目盲。除了显而易见的效果外，目盲的生物在战斗中有50%的失手率（所有敌人都具有全隐蔽），失去AC上的敏捷加值，并且AC受到-2减值，同时依赖视觉的察觉检定以及大多数基于力量和敏捷的技能检定受到-4的减值。黑暗区域包括未点燃的地下城房间、大多数洞穴，以及阴云密布、无月的夜晚。

具有昏暗视觉的角色（如精灵、侏儒和半精灵）可以看到两倍于给定光照半径距离的物体。对于这些角色，明亮光照、普通光照和昏暗光照的有效半径加倍。

具有黑暗视觉的角色（如矮人和半兽人）可以正常看到光照区域，并且能够看到60英尺内的黑暗区域。在60英尺范围内，除非生物是隐形的或有掩蔽，否则无法对具有黑暗视觉的角色隐藏。

## 破坏物品

:::origin[[英文原文](https://aonprd.com/Rules.aspx?ID=125)]
:::

当试图破坏一个物体时，你有两种选择：用武器将其砸碎，或凭借纯粹的力量将其打破。

### 以攻击破坏物体

用挥砍或钝击武器砸碎武器或盾牌，可以通过破武战技实现（见第8章）。砸碎物体与破武武器或盾牌类似，不同之处在于你的战技检定是与物体的AC对抗的。通常，你只能使用钝击或挥砍武器来砸碎物体。

**护甲等级**： 物体比生物更容易命中，因为它们通常不会移动，但许多物体足够坚固，可以抵挡住每次打击的一部分伤害。物体的护甲等级等于10+其体型调整值（见表7-11）+其敏捷调整值。一个无生命的物体不仅具有0的敏捷值（因此AC会有-5的减值），还会有额外的-2AC减值。此外，如果你使用整轮动作来瞄准，可以用近战武器自动命中，使用远程武器时攻击检定获得+5加值。

:::table[表7-11 体型和物品AC]
| 体型 | AC调整值 |
| :---- | :----: |
| 超巨型 | -8 |
| 巨型 | -4 |
| 超大型 | -2 |
| 大型 | -1 |
| 中型 | +0 |
| 小型 | +1 |
| 超小型 | +2 |
| 微型 | +4 |
| 超微型 | +8 |
:::

**硬度**：每个物体都有硬度——这是一个表示其抗伤害能力的数值。当物体受到伤害时，从伤害值中减去其硬度。只有超过其硬度的伤害才会从物体的生命值中扣除（见表7-12、表7-13和表7-14）。

**生命值**：物体的生命值总数取决于它的材质和大小（见表7-12、表7-13和表7-14）。受到等于或大于其总生命值一半的伤害的物体会获得“破损”状态（见附录2）。当物体的生命值降至0时，它就会被毁坏。

非常大的物体不同部分有各自的生命值总数。

*能量攻击*：能量攻击对大多数物体造成一半的伤害。在应用物体的硬度之前，将伤害值减半。某些能量类型可能对特定物体特别有效，这取决于GM的判断。例如，火焰可能对羊皮纸、布料和其他易燃物体造成全额伤害。音波可能对玻璃和水晶物体造成全额伤害。

*远程武器伤害*：物体受到远程武器的伤害减半（除非武器是攻城器械或类似的武器——见第434页）。在应用物体的硬度之前，将造成的伤害值减半。

*无效武器*：某些武器无法有效地对某些物体造成伤害。例如，钝击武器不能用于损坏绳索。同样，大多数近战武器对石墙和石门几乎没有效果，除非它们是用于破石的工具，如镐或锤子。

*免疫*：物体免疫非致命伤害和重击。

*魔法护甲、盾牌和武器*：每+1的增强加值会为护甲、武器或盾牌增加2点硬度，并为该物品增加10点生命值。

*对某些攻击易伤*：某些攻击对特定物体特别有效。在这种情况下，攻击会造成双倍伤害，并可能无视物体的硬度。

*受损物体*：受损的物体在获得“破损”状态后仍然可以使用，直到其生命值减少到0，此时它会被摧毁。受损但未被摧毁的物体可以通过工艺技能和一些法术进行修复。

**豁免检定**：非魔法的无主物品物品从不进行豁免检定。它们被视为未通过豁免检定，因此在面对允许进行豁免检定以抵抗或抵消的法术和其他攻击时，它们总是会完全受到影响。角色的物品（被握持、触碰或佩戴）会使用角色的豁免检定（即，使用角色的豁免检定加值）。

魔法物品总是可以进行豁免检定。魔法物品的强韧、反射和意志豁免加值等于2+其施法者等级的一半。角色的魔法物品可以选择使用其主人的豁免检定，或者使用自己的豁免检定加值，以较高者为准。

*活化物体*：活化物体在确定其护甲等级时被视为生物（不将其视为无生命物体）。

### 以力量破坏物品

当角色试图通过突然的力量而非造成伤害来破坏或破裂某物时，应使用力量检定（而不是像破武战技那样使用攻击检定和伤害检定）来决定他是否成功。由于硬度不会影响物品的破坏DC（难度等级），因此这个值更多地取决于物品的结构而非制造该物品的材料。参考表7-15以获取常见的破坏DC列表。

如果某个物品失去了其一半或更多的生命值，该物品将获得破损状态（参见附录2），且其破坏DC将降低2。

较大和较小的生物在力量检定中根据体型对破坏门的检定获得体型加值或体型减值，具体如下：超微型-16，微型-12，超小型-8，小型-4，大型+4，超大型+8，巨型+12，超巨型+16。

撬棍或便携式攻城锤可以提高角色破门的几率（参见第6章）。
