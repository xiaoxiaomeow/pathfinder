# 第二章 种族

:::info[编注]
每个种族的细节被统一整理，参见TBA。
:::

从健壮的矮人到高贵的精灵，Pathfinder角色扮演游戏的种族是一个由多种文化、体型、态度和外貌组成的多样化混合体。在你生成角色的基本属性值后，角色创建过程的下一步就是选择你的角色种族；本章节提供了七种不同的选择。这七个种族组成了Pathfinder RPG中最常见的文明种族。

选择你的角色种族是你需要做出的较为重要的决定之一。随着你的角色变得更强大，你将能够通过选择不同的职业、技能和专长来多样化他的能力，但你只能选择一次种族（除非有一些不寻常的魔法，比如[转生术](/pathfinder/spell?spell=reincarnate)）。当然，每个种族最适合特定类型的角色——比起术士，矮人更适合成为战士，而半身人不如半兽人适合成为野蛮人。在做出选择时，请记住每个种族的优劣势。虽然逆势而为地扮演一个种族也很有趣，但在角色达到三级时才意识到你想玩的角色本来应该是另一种族，这就不那么有趣了。

本章节中的每个种族都以相同的格式呈现，首先是该种族在世界中的一般描述。接下来是对该种族普通成员的体貌描述、该种族社会的简要概述以及该种族与其他六个种族的关系。尽管你的种族不会强制你选择某一种宗教或阵营，但会提到每个种族的典型选择。接下来讨论的是该种族成员为什么可能决定选择充满危险的冒险生活。最后，我们列出了每个种族的男性和女性的一些示例名字。

本章节中的每个种族都有一套特殊的能力、加值和适用于所有成员的其他调整。这些是你的角色的“种族特性”。每个种族也有在你生成属性值后应用的属性值调整值，如前一章所述。这些调整值可以将属性值提高到18以上或降低到3以下——尽管在你的任何属性中有如此低的数值是你应该避免的，因为没有比低体质更能确保角色死亡的途径，也没有比智力低于3的角色更让人沮丧的情况。你应该在扮演任何属性值低于3的角色前征得你的GM的批准。

本章节介绍的七个种族有着截然不同的能力、个性和社会，但同时，这七个种族又非常相似——这里没有任何种族与人类相差太远，他们的能力都大致相等并且平衡。在游戏世界中还存在着其他更强大和更异域的种族，但Pathfinder RPG是建立在所有玩家起点大致相等的基础上的。有关扮演更强大或更不寻常种族的规则和指南，请参见第12章。
