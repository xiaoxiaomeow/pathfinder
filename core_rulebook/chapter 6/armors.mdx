# 防具

:::origin[[英文原文](https://www.aonprd.com/Rules.aspx?ID=361)]
:::

:::info[译注]
似乎中文社区中，护甲或盔甲通常指穿在身上的护甲，盾牌是绑在手臂上的盾牌，防具通常是护甲和盾牌的统称。但并不是所有译者都按照这种方式翻译，而英文原文中，护甲/盔甲和防具都是Armor，因此遇到歧义请务必查看原文并按照语境判断。
:::

对于大多数人来说，在充满威胁和危险的世界中，防具是最简单的自我保护方式。许多角色只能穿戴最简单的护甲，只有一部分人能够使用盾牌。要有效穿戴较重的护甲，角色可以选择擅长盔甲专长，但大多数职业自动擅长使用最适合他们的防具。

以下是防具条目的格式（在表6-6中作为列标题给出）。

**价格**：小型或中型类人生物的防具价格，以金币计算。其他生物的防具价格见[表6-8](#特殊生物的防具)。

**护甲/盾牌加值**：每种护甲都会对AC（护甲等级）提供护甲加值，而盾牌会对AC提供盾牌加值。一套护甲的护甲加值不会与其他效果或物品提供的护甲加值叠加。同样，盾牌的盾牌加值不会与其他效果提供的盾牌加值叠加。

**最大敏捷加值**：这个数字是此类护甲允许的AC的最大敏捷加值。在计算穿戴者的AC时，超过这个数字的敏捷加值会被减少到这个数字。较重的护甲限制了移动性，减少了穿戴者闪避攻击的能力。这个限制不会影响任何其他与敏捷相关的能力。

即使角色因为护甲的原因导致其AC的敏捷加值降至0，这种情况也不算作失去敏捷加值。

角色的负重（包括护甲在内的装备量）也可能限制可应用于护甲等级的最大敏捷加值。

*盾牌*：盾牌不会影响角色的最大敏捷加值，除了塔盾。

**防具检定减值**：任何比皮甲重的护甲，以及任何盾牌，都会削弱角色使用基于敏捷和力量的技能的能力。防具检定减值适用于所有基于敏捷和力量的技能检定。角色的负重也可能导致防具检定减值。

*盾牌*：如果角色穿戴护甲并使用盾牌，两个防具检定减值都适用。

*不擅长所穿防具*：穿戴不擅长的护甲和/或使用不擅长的盾牌的角色，在攻击掷骰以及所有基于敏捷和力量的属性和技能检定上都会受到护甲（和/或盾牌）的防具检定减值。不擅长护甲的减值与不擅长盾牌的减值叠加。

*穿着护甲睡觉*：穿着中型或重型护甲睡觉的角色第二天会自动疲劳。他在力量和敏捷上会受到-2的减值，且不能进行冲锋或奔跑。穿着轻型护甲睡觉不会导致疲劳。

**奥术失败几率**：防具会干扰施法者施展包含姿势成分的奥术法术。奥术施法者在穿戴防具时面临奥术失败的可能性。吟游诗人可以穿轻型护甲并使用盾牌而不对其吟游诗人法术造成奥术失败几率。

*穿着护甲施展奥术法术*：角色在穿戴护甲时施展奥术法术通常需要进行奥术失败率检定。表6-6中的“奥术失败率”一栏中的数字是法术失败和被破坏的百分比几率。然而，如果法术不包含姿势成分，则可以无奥术失败率地施展。

*盾牌*：如果角色穿戴护甲并使用盾牌，将两个数字相加得到单一的奥术失败几率。

**速度**：中型或重型护甲会减慢穿戴者的速度。表6-6中的数字是角色穿戴护甲时的速度。人类、精灵、半精灵和半兽人的未负重速度为30英尺。他们使用第一列。矮人、侏儒和半身人的未负重速度为20英尺。他们使用第二列。然而，矮人的陆地速度即使在穿着中型或重型护甲或携带中型或重型负重时仍然是20英尺。

*盾牌*：盾牌不会影响角色的速度。

**重量**：此列给出了适合中型使用者的防具重量。适合小型角色的防具重量减半，适合大型角色的防具重量加倍。

## 精制品防具

与武器一样，你也可以购买或制作精制品版本的护甲或盾牌。这样的精良物品与普通版本功能相同，但其防具检定减值减少1。

一套精制品护甲或盾牌比相同类型的普通护甲或盾牌额外多花费150gp。

即使护甲或盾牌被用作武器，精制品护甲或盾牌的品质也不会提供攻击或伤害加值。

所有的魔法护甲和盾牌自动被视为具有精制品质量。

你不能在护甲或盾牌制作完成后再添加精制属性；它必须在制作时就成为精制品物品。

## 穿戴和脱下防具

穿戴防具所需的时间取决于其类型；见表6-7。

**穿戴**：本列说明了角色穿上防具所需的时间。（一分钟等于10轮。）准备（绑上）盾牌只需一个移动动作。

**匆忙穿戴**：本列说明了在匆忙中穿上护甲所需的时间。匆忙穿戴的护甲在防具检定减值和护甲加值上比正常情况各差1点。

**脱下**：本列说明了脱下防具所需的时间。从手臂上移除并丢弃盾牌只需一个移动动作。

:::table[表6-7 穿戴防具]
| 防具类型 | 穿戴 | 匆忙穿戴 | 脱下 |
| :---- | :----: | :----: | :----: |
| 任意盾牌 | 1个移动动作 | n/a | 1个移动动作 |
| 护垫甲，皮甲，革甲，镶嵌皮甲或链甲衫 | 1分钟 | 5轮 | 1分钟<sup>1</sup> |
| 胸甲，鳞甲，链甲，混织铁甲或板条甲 | 4分钟<sup>1</sup> | 1分钟 | 1分钟<sup>1</sup> |
| 半身甲或全身甲 | 4分钟<sup>2</sup> | 4分钟<sup>1</sup> | 1d4+1分钟<sup>1</sup> |

1 如果角色有协助，则将穿戴时间减半。单个角色在没有其他任务的情况下可以协助一到两个相邻的角色。两个人不能同时互相协助穿戴护甲。

2 穿戴者必须有协助才能穿上这种护甲。如果没有协助，只能匆忙穿戴。
:::

:::info[译注]
盾牌并不是从包里拿出来就可以提供盾牌加值，它们都需要再用一个移动动作“穿戴和脱下防具”被绑在手臂上，才能提供盾牌加值。拆下来并丢掉盾牌则需要一个移动动作“穿戴和脱下防具”和一个自由动作扔下物品。
:::
