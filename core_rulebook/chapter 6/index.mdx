# 第六章 装备

:::origin[[英文原文](https://www.aonprd.com/Rules.aspx?ID=105)]
:::

:::info[编注]
每个装备的细节被统一整理，参见TBA。
:::

装备精良的角色几乎可以应对任何挑战，从在荒野中生存到在国王的宴会上留下深刻印象。本章介绍了各种普通和奇异的装备供PC们购买和使用，包括武器、护甲、炼金物品、精制品工具、美酒以及野外口粮。这里介绍的装备在大多数城镇和城市应该相对容易找到并购买，尽管GM可能希望限制一些更昂贵和奇异物品的供应。魔法物品则难得多（见第15章）。
