# 学习技能

:::origin[[英文原文](https://www.aonprd.com/Rules.aspx?ID=86)]
:::

每个等级，角色根据其职业和智力调整值获得一定数量的技能点数。向一个技能投入的一个技能点代表对该技能的一定训练程度。你在一个技能中的点数不能超过你总的生命骰数。此外，每个职业都有一些偏爱的技能，称为本职技能。角色更容易在这些技能中变得更精通，因为它们代表了他职业训练和不断练习的一部分。你在所有投入了点数的本职技能上获得+3加值。如果你有多个职业，并且它们都给予你本职技能加值，这些加值不叠加。

当你在基础职业中提升等级时，获得的技能点数见表4–1。人类每个职业等级获得1个额外技能点。选择天赋职业的角色可以选择获得1个额外技能点或1个额外的生命值（参见[第31页](/pathfinder/core_rulebook/chapter%203/character%20advancement#天赋职业)）。如果你选择了一个新职业，该职业的所有本职技能自动添加到你的本职技能列表中，如果你在这些技能中投入过点数，你将在这些技能上获得+3加值。

:::table[表4-1 技能点数量]
| 职业 | 每等级技能点数量 |
| :---- | :----: |
| 野蛮人 | 4 + 智力调整值 |
| 吟游诗人 | 6 + 智力调整值 |
| 牧师 | 2 + 智力调整值 |
| 德鲁伊 | 4 + 智力调整值 |
| 战士 | 2 + 智力调整值 |
| 武僧 | 4 + 智力调整值 |
| 圣武士 | 2 + 智力调整值 |
| 游侠 | 6 + 智力调整值 |
| 盗贼 | 8 + 智力调整值 |
| 术士 | 2 + 智力调整值 |
| 法师 | 2 + 智力调整值 |
:::

## 技能检定

当角色使用技能时，不保证一定会成功。为了确定成功与否，每当你尝试使用技能时，你必须进行技能检定。每个技能点数在使用该技能进行检定时提供+1加值。当你进行技能检定时，你掷1d20，然后将你的技能点数和相应的属性调整值加到检定结果中。如果你使用的技能是本职技能（并且你在该技能上投入了点数），你在检定时获得+3加值。如果你没有受过该技能的训练（且该技能可以未经训练使用），你仍可以尝试使用该技能，但只使用相关属性调整值提供的加值（或减值）来调整检定。技能还可以通过多种来源进一步调整——例如种族、职业能力、装备、法术效果或魔法物品等。有关技能检定加值的总结，请参见表4–2。

:::table[表4-2 技能检定加值]
| 技能 | 技能检定等于* |
| :---- | :---- |
| 未受训 | 1d20 + 属性调整值 + 种族调整值 |
| 受训 | 1d20 + 技能等级 + 属性调整值 + 种族调整值 |
| 受训本职技能 | 1d20 + 技能等级 + 属性调整值 + 种族调整值 + 3 |

\* 防具检定减值应用到所有基于力量和敏捷的技能检定。
:::

如果你的技能检定结果等于或大于你尝试完成的任务的难度等级（或DC），你就成功了。如果结果低于DC，你就失败了。一些任务根据你的检定结果高于或低于要求的DC的程度，具有不同的成功和失败级别。一些技能检定是由目标的技能检定对抗的。在进行对抗技能检定时，如果你的检定结果超过目标的结果，尝试就成功。

## 取10和取20

一次技能检定代表了一次完成某个目标的尝试，通常在某种时间压力或干扰下进行。然而，有时候角色可以在更有利的条件下使用技能，从而增加成功的几率。

**取10**：当你的角色不在立即危险或干扰中时，你可以选择取10。取10时，不掷1d20进行技能检定，而是将结果视为你掷出了10。对于许多常规任务，取10会使它们自动成功。干扰或威胁（如战斗）使角色无法取10。在大多数情况下，取10纯粹是一种安全措施——你知道（或预期）一个平均的掷骰结果会成功，但担心一个差的掷骰结果会失败，所以你选择接受平均结果（10）。取10在某些情况下特别有用，因为特别高的掷骰结果不会有帮助。

**取20**：当你有充足的时间，没有面临任何威胁或干扰，并且所尝试的技能没有失败惩罚时，你可以取20。换句话说，如果你掷d20足够多次，最终你会掷出20。取20时，不掷1d20进行技能检定，而是将结果视为你掷出了20。

取20意味着你不断尝试直到成功，并假设你在成功前失败了很多次。取20需要的时间是进行一次单次检定所需时间的20倍（通常对于一个需要1轮或更少时间执行的技能，需要2分钟）。

由于取20假设你的角色在成功前会失败很多次，因此你的角色会自动遭受任何失败的惩罚（因此通常不允许对有失败惩罚的技能使用取20）。常见的“取20”技能包括解除装置（用于开锁时）、逃脱和察觉（尝试寻找陷阱时）。

**属性检定和施法者等级检定**：取10和取20的常规规则适用于属性检定。这两种规则都不适用于专注检定或施法者等级检定。

## 援助他人

你可以通过进行相同类型的技能检定来帮助别人成功进行技能检定，这是一种合作努力。如果你在检定中掷出10或更高的结果，你所帮助的角色在他的检定中获得+2加值（你不能在帮助他人的技能检定中取10）。在许多情况下，一个角色的帮助可能无益，或者只能有有限数量的角色同时提供帮助。

在某些情况下，技能限制了谁可以取得某些结果，例如尝试使用解除装置开锁，你不能帮助他人获得一个你的角色无法单独完成的任务加值。GM可能会根据具体情况对帮助他人施加进一步的限制。
