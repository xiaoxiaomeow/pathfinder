# 技能描述

:::origin[[英文原文](https://www.aonprd.com/Rules.aspx?ID=341)]
:::

本节描述了每项技能，包括常见的用途和典型的调整值。角色有时可以在GM的裁量下将技能用于这里未提到的其他目的。有关所有技能的完整总结，请参见表4-3。

技能描述遵循以下格式：

**技能名称**：技能名称行除了技能名称外，还包括以下信息。

**关键属性**：适用于技能检定的属性调整值的缩写。

**需受训**：如果在技能名称行中包含此标注，则必须在该技能中至少有1级才能使用。如果没有此标注，则该技能可以在未受训（等级为0）的情况下使用。如果有任何特殊说明适用于受训或未受训使用，它们将在未受训部分（见下文）中涵盖。

**防具检定减值**：如果在技能名称行中包含此标注，则防具检定减值适用于使用该技能的检定（见第六章）。如果没有此标注，则防具检定减值不适用。

**描述**：技能名称行后是对使用该技能代表什么的一般描述。

**检定**：角色（技能描述中的“你”）在成功的技能检定中可以做什么，以及检定的难度等级（DC）。

**动作**：使用该技能所需的动作类型或所需的时间。

**重试**：任何适用于成功尝试再次使用该技能的条件。如果技能不允许你多次尝试同一任务，或如果失败带有固有的惩罚（例如攀爬技能），则不能取20。如果省略此段落，则技能可以在没有固有惩罚的情况下重试，除了所需的额外时间。

**特殊**：适用于该技能的任何额外事实，例如使用该技能产生的特殊效果，或由于职业、专长选择或种族而获得的加值。

**限制**：某些技能的全部效用仅限于某些职业的角色。此项指出该技能是否存在任何此类限制。

**未受训**：此项说明没有在该技能中至少有1级的角色可以用它做什么。如果没有此项，则表示该技能对未受训角色（如果可以未受训使用）正常运作，或者未受训角色不能尝试该技能的检定（对于标有“需受训”的技能）。
