# 第三章 职业

:::info[编注]
每个职业的细节被统一整理，参见TBA。
:::

角色的职业是他最具定义性的特征之一。它是他大多数能力的来源，并在任何冒险队伍中给予他特定的角色。以下十一种职业代表了游戏的核心职业。

**野蛮人**：野蛮人是来自文明边缘之外的残暴狂战士。

**吟游诗人**：吟游诗人同时使用技能和法术来增强他的盟友，迷惑他的敌人，并提升他的声誉。

**牧师**：牧师是虔诚的神明追随者，能够治疗伤口、复活死者，并召唤神明的愤怒。

**德鲁伊**：德鲁伊崇拜自然万物——他是施法者、动物的朋友和熟练的变形者。

**战士**：勇敢坚定，战士是所有武器和盔甲的高手。

**武僧**：武僧是武术的学生，他训练自己的身体成为最强的武器和防御。

**圣武士**：圣武士是披着闪亮盔甲的骑士，忠实于法律和善良的追随者。

**游侠**：游侠是追踪者和猎人，是野外生物和追踪其钟爱敌人的专家。

**盗贼**：盗贼是小偷和侦察者，是能够对毫无防备的敌人进行残酷打击的机会主义者。

**术士**：施法的术士天生就具备魔法天赋，拥有奇异的神秘力量。

**法师**：法师通过不断的学习掌握魔法，从而获得令人难以置信的魔法力量。
