# 角色升级

:::origin[[英文原文](https://www.aonprd.com/Rules.aspx?ID=85)]
:::

当玩家角色克服挑战时，他们会获得经验值。随着这些点数的积累，PC会在等级和力量上进步。这种进步的速度取决于你的团队想要进行的游戏类型。有些人喜欢快节奏的游戏，在这种游戏中，角色每进行几次游戏就会提升等级，而另一些人则喜欢进步较慢的游戏。最终，由你的团队决定哪种速度最适合你们。角色根据表格3–1提升等级。

## 升级你的角色

当角色获得足够的经验值时，他的等级就会提升——通常，这是在一次游戏结束时，当你的GM（游戏主持人）发放该次游戏的经验值奖励时发生的。

提升角色的过程与生成角色的方式非常相似，只不过你的属性值、种族和之前关于职业、技能和专长的选择不能改变。提升等级通常会给你带来新能力、额外的技能点数、更多的生命值、可能还有你选择的一个属性值的永久+1加值或一个额外的专长（参见表格3–1）。随着时间的推移，随着你的角色等级升高，他将成为游戏世界中真正强大的力量，能够统治国家或使其屈服。

在添加现有职业的新等级或添加新职业的等级时（见下文的多职业），请确保按以下步骤进行。首先，选择你的新职业等级。在进行以下任何调整之前，你必须能够满足该等级的要求。第二，应用由于提升等级而获得的任何属性值加值。第三，整合该等级的所有职业能力，然后掷骰子增加生命值。最后，添加新技能和专长。

有关何时获得新专长和属性值提升的更多信息，请参见表格3–1。

## 兼职

与其获得角色当前职业下一级的能力，他可以选择获得新职业的1级能力，将所有这些能力添加到他现有的能力中。这被称为“兼职”。

例如，假设一个5级战士决定涉猎奥术，并在提升到6级时添加一个法师的等级。这样的角色将拥有5级战士和1级法师的职业特性和能力，但仍被视为6级角色。（他的职业等级分别是5级和1级，但他的总角色等级是6级。）他保留从5级战士中获得的所有奖励专长，但现在也可以施展1级法术并选择一个奥术学派。他将1级法师的所有生命值、基本攻击加值和豁免加值添加到5级战士的这些值之上。

## 天赋职业

每个角色开始游戏时可以选择一个天赋职业——通常，这是他在1级时选择的职业。每当角色在其天赋职业中获得一个等级时，他会获得+1生命值或+1技能点。天赋职业的选择在角色创建后不能更改，每次角色升级（包括第一级）时获得生命值或技能点的选择一旦做出便不能更改。进阶职业（参见第11章）永远不能成为天赋职业。
