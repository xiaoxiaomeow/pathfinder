# 专长描述

专长在表5–1中进行了总结。

请注意，此表中的专长的先决条件和专长效果已简略，以便参考。有关详细信息，请参阅专长描述。

所有专长描述均使用以下格式。

**专长名称**：专长名称还指示了该专长所属的子类别（如果有），并附有该专长作用的基本描述。

**先决条件**：获得专长所需的最低属性值、其他专长、最低基础攻击加值、一种或多种技能的最低等级，或其他任何要求。如果专长没有先决条件，则此项省略。一个专长可能有多个先决条件。

**专长效果**：该专长使角色（专长描述中的“你”）能够做什么。如果一个角色多次拥有同一个专长，除非描述中另有说明，否则其专长效果不能叠加。

**通常状况**：没有此专长的角色所受到的限制。如果没有专长不会导致特别的缺陷，则此项省略。

**特殊说明**：关于该专长的额外特殊信息。
