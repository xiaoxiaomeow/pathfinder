# 第五章 专长

:::origin[[英文原文](https://www.aonprd.com/Feats.aspx)]
:::

:::info[编注]
每个专长的细节被统一整理，参见[专长速查](/pathfinder/feats)。
:::

有些能力与种族、职业或技能无关——比如特别快的反应让你能更迅速地应对危险、制作魔法物品的能力、使用近战武器进行强力攻击的训练，或是偏转射向你的箭矢的技巧。这些能力以专长的形式体现。虽然有些专长对某些类型的角色比对其他角色更有用，并且许多专长在选择前必须满足特殊的前提条件，但一般来说，专长代表了你角色种族和职业之外的能力。许多专长会改变或增强职业能力或减轻职业限制，而其他专长可能会为你的属性提供加值或授予你本来无法进行的行动能力。通过选择专长，你可以个性化和调整你的角色，使其独一无二。
