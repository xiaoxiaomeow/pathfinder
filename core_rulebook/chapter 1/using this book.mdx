# 使用本书

本书分为15章，还有一些附录。第1章到第11章涵盖了玩家创建角色和玩游戏所需的所有规则。第12章到第15章包含帮助游戏主持人运行游戏和裁定世界的信息。一般来说，如果你是玩家，不需要了解这些后面章节的信息，但有时可能会参考它们。以下是这些章节的简介，旨在为你提供本书中规则的概述。

**第1章（开始游戏）：** 本章介绍了Pathfinder角色扮演游戏的基础，包括如何参考本书的其他部分、生成玩家角色（PC）的规则以及确定角色属性值的规则。属性值是角色最基本的属性，描述了他的潜力和能力。

**第2章（种族）**： Pathfinder角色扮演游戏包含七个核心种族，代表了游戏世界中最常见的种族。它们是矮人、精灵、侏儒、半精灵、半兽人、半身人和人类。本章涵盖了扮演这些种族成员所需的所有规则。在创建PC时，你应该选择本章中的一个种族。

**第3章（职业）**： Pathfinder角色扮演游戏有11个核心职业。职业代表角色的基本职能，每个职业都赋予一系列特殊能力。角色的职业还决定了角色使用的一系列其他统计，包括生命值、豁免加值、武器和护甲擅长及技能点数。本章还涵盖了角色随着能力的增长（等级提升）而进阶的规则。在创建PC时，你应该从本章中选择一个职业，并将一个等级放入该职业（例如，如果你选择你的起始职业为法师，你将是1级法师）。

**第4章（技能）**： 本章介绍了技能以及如何在游戏中使用它们。技能代表角色可以执行的各种简单任务，从攀爬墙壁到偷偷溜过守卫。每个角色会获得一定数量的技能点数，这些点数可以用来让角色更擅长使用某些技能。随着角色等级的提升，他会获得额外的技能点数，这些点数可以用来提高角色已经拥有的技能或掌握新技能。角色的职业决定了角色可以分配多少技能点数。

**第5章（专长）**： 每个角色都拥有一些专长，这些专长允许角色执行某些特殊动作或赋予一些其他通常无法获得的能力。每个角色开始游戏时至少拥有一个专长，随着角色等级的提升会获得新的专长选择。

**第6章（装备）：** 本章介绍了可以购买的基本装备，从护甲和武器到火把和背包。这里还列出了常见服务的费用，例如住旅馆或乘船的费用。初始角色根据他们各自的职业获得一定数量的金币，可以在1级时用来购买装备。

**第7章（额外规则）：** 本章涵盖了几条与玩Pathfinder角色扮演游戏重要的杂项规则，包括阵营、负重、移动和可见度。阵营告诉你你的角色是无可救药的反派、正义的英雄还是介于两者之间的任何位置。负重处理你的角色在不受阻碍的情况下可以携带多少重量。移动描述了你的角色在一分钟、一小时或一天内可以行进的距离，这取决于他的种族和环境。可见度处理你的角色根据种族和现有光线条件可以看到多远。

**第8章（战斗）：** 所有角色最终都会与可怕的怪物和危险的反派进行生死搏斗。本章介绍了如何在Pathfinder角色扮演游戏中处理战斗。在战斗中，每个角色按顺序行动（由先攻决定），顺序会重复直到一方死亡或被击败。在本章中，你会找到战斗中每回合可以执行的各种动作的规则。本章还包括裁定特殊战斗动作的规则（例如试图绊倒敌人或试图解除敌人的武器）以及角色受伤和死亡的规则。

**第9章（魔法）：** 许多职业（以及一些怪物）可以施放法术，这些法术几乎可以做到任何事情，从让死人复生到用火球烧死敌人。本章处理施放法术和学习新法术的规则。如果你的角色可以施放法术，你应该熟悉这些规则。

**第10章（法术）：** 魔法章节描述了如何施放法术，本章则处理各个法术本身，从按职业分类的可用法术列表开始。随后是游戏中每个法术的详细列表，包括其效果、范围、持续时间和其他重要变量。能够施放法术的角色应该阅读所有可用的法术。

**第11章（进阶职业）：** 尽管第3章中的核心职业允许多种角色类型，但进阶职业使角色能够成为某一特定主题的专家。这些高级职业提供了一系列专门的能力，使角色在某一领域非常强大。角色在选择进阶职业等级之前必须满足特定的前提条件，这些前提条件因进阶职业而异。如果你计划在进阶职业上投入等级，应该熟悉这些前提条件，以确保你的角色最终能满足这些要求。

**第12章（运行游戏）：** 本章涵盖了运行Pathfinder角色扮演游戏的基本知识，包括创建游戏、使用已发布的冒险模组、在桌面上裁决事务以及奖励经验点数和宝藏的指南。如果你是游戏主持人（GM），你应该熟悉本章介绍的概念。

**第13章（环境）：** 除了对抗怪物，玩家角色在玩Pathfinder角色扮演游戏时还会遇到许多其他危险和挑战。本章涵盖了裁决环境规则的指南，从狡诈的陷阱到沸腾的岩浆，并按环境类型分解，包括地牢、沙漠、山脉、森林、沼泽、水域、城市以及超现实的其他维度和位面。最后，本章还包括天气及其对游戏影响的信息。

**第14章（创建NPC）：** 除了玩家角色和怪物，游戏世界中还有无数的非玩家角色（NPC）。这些角色由GM创建和控制，代表游戏世界中存在的所有其他人物，从当地的商店老板到贪婪的国王。本章包括大多数NPC使用的简单职业（尽管有些NPC可能拥有核心职业和进阶职业的等级）以及快速生成NPC统计数据的系统。

**第15章（魔法物品）：** 角色在冒险过程中经常会找到魔法物品来帮助他们克服困难。本章详细介绍了这些魔法物品，包括武器、护甲、药水、戒指、权杖、卷轴、法杖和奇物（一个涵盖其他所有物品的通用类别）。此外，你还会在本章中找到诅咒物品（阻碍持有者的物品）、智能物品、神器（具有极大力量的物品）以及创造新魔法物品的规则。

**附录：** 书末的附录汇集了许多关于特殊能力和状态的单独规则。本节还包括推荐阅读书目，以及讨论其他工具和产品的信息，以提升Pathfinder角色扮演游戏的体验。
